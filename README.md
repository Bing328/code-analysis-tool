## 🔐 安全要求

### 监控堆栈部署
部署监控堆栈时，必须通过环境变量设置密码：
```bash
# 设置密码环境变量
export GRAFANA_ADMIN_PASSWORD="YourSecurePassword123!"

# 运行部署脚本
sudo ./setup_monitor_v2.sh
密码要求
至少12位长度

包含大小写字母、数字和特殊字符

不能包含常见弱密码模式

安全最佳实践
不要将密码硬编码在脚本中

使用密码管理器生成和存储密码

定期轮换重要服务的密码

确保环境变量文件不被提交到版本控制
